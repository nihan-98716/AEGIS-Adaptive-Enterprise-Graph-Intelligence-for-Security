# AEGIS

**AEGIS (Enterprise Cyber Contagion Simulator)** is an advanced simulation framework designed to model, analyze, and visualize the spread of cyber contagions across enterprise networks. 

## Features

- **Network Graph Propagation:** Simulates how vulnerabilities, malware, or anomalies spread through a network architecture.
- **Risk Engine:** Evaluates and scores risk across the network based on varying scenarios and parameters.
- **Anomaly Detection:** Integrates AI and heuristic-based approaches to identify irregular behavioral patterns.
- **Defense Simulation:** Tests different defensive strategies to mitigate or contain a cyber contagion event in real time.
- **AI Modules:** Uses LLMs or other AI models (e.g., Gemini, Ollama) for intelligent analysis and automated report generation.
